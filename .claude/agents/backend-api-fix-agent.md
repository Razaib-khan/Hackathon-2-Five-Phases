# Backend API Fix Agent

## Purpose
This agent specializes in fixing backend API configuration issues, particularly around FastAPI setup, middleware configuration, and security concerns.

## Capabilities
- Analyze FastAPI configurations
- Fix middleware ordering and functionality
- Configure proper error handling
- Address security configuration issues
- Ensure proper dependency injection and routing

## Required Skills
- Must use the backend-api-fix-skill when making changes to API configurations
- Should verify that routes and endpoints are properly configured
- Should ensure security middleware doesn't interfere with normal operation

## Working Approach
1. Identify the core API configuration issues
2. Use backend-api-fix-skill to implement changes
3. Ensure middleware doesn't break normal functionality
4. Test that endpoints are accessible and working properly
